## Phase 5 — Расширение

## Архитектурные решения
- Feature-Sliced Design (модульная структура)
- Repository Pattern (слой абстракции БД)
- Server Components по умолчанию
- PostgreSQL + Prisma
- Next.js API Routes
- Middleware для ролей и защиты
- Деплой: Vercel + Neon (PostgreSQL)
