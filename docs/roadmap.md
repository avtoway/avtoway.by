# Roadmap — АВТОWAY

## Phase 1: Multi-page site (текущая задача)

### Страницы
- [ ] **Главная** (есть, доработать)
- [ ] **Услуги** — список услуг с описанием
- [ ] **О нас** — информация о проекте, команде
- [ ] **Контакты** — форма обратной связи, email, соцсети
- [ ] **Партнёры** — информация для партнёров
- [ ] **Политика конфиденциальности** (обязательно)

### База данных
- [ ] Выбрать БД (PostgreSQL)
- [ ] Настроить Prisma ORM
- [ ] Создать схему данных:
  - Services (услуги)
  - Contacts (заявки с формы)
  - Partners (партнёры)
  - Settings (настройки сайта)

### Админка (без регистрации пользователей)
- [ ] Отдельный маршрут `/admin`
- [ ] Защита паролем (не регистрация)
- [ ] CRUD для услуг
- [ ] Просмотр заявок с формы
- [ ] Редактирование контента страниц
- [ ] Управление партнёрами

### Роли и доступ (без регистрации)
- [ ] **Admin** — полный доступ
- [ ] **Editor** — управление контентом (без настроек)
- [ ] **Partner** — доступ только к своему разделу
- [ ] Реализация через один логин + middleware

### Контент
- [ ] Придумать структуру услуг
- [ ] Написать тексты для страниц
- [ ] Собрать ссылки на все соцсети (YouTube, Instagram, Telegram, VK и т.д.)

## Phase 2: Расширение функционала
