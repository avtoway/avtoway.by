## Открытые

### [BUG-001] Карусель — ховер на пиках обрезается
**Дата:** 2026-07-09
**Приоритет:** Средний
**Компонент:** `src/shared/ui/carousel.tsx`, `src/components/services-carousel.tsx`
**Описание:** При наведении на карту в позиции пика тень (`box-shadow: 0 20px 60px`) обрезается `overflow: hidden`. Маска (`mask-image`) сглаживает край, но тень всё равно видна только в пределах 64px пика.
**Попытки исправить:**
  1. `overflow: visible` + градиент-оверлей — тень вылезала за пик
  2. Абсолютный shadow-слой + React state — артефакты, прыжки
  3. `box-sizing: border-box` + padding — сломал высоту
  4. Уменьшение blur тени (30px вместо 60px) — не понравилось
**Вывод:** Фундаментальное ограничение CSS. `overflow: hidden` всегда обрезает `box-shadow` детей. `mask-image` НЕ решает проблему (маска тоже обрезает тень). Оставлено как есть — тень видна в пределах пика, край сглажен маской.
